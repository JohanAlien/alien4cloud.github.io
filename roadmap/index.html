Roadmap

The following represents current view of Alien4Cloud development team of its product development cycle and future directions. It is intended for information purposes only, and should not be interpreted as a commitment, though we do our best to reach the dates and features set mentioned below.

Premium features, support and certifications are available under a subscription, delivered by FastConnect, a Bull, Atos technologies subsidiary. Contact us for more details.

We deliver major releases every 4 month. Please note that we also deliver intermediate sprint releases in Alien4Cloud GitHub repo, every 3 weeks, in order to get feedback on features while still in development.

  • November 2016 v 1.3.0

    Core

    • Simplified API
    • New version of the designer
      • Advanced Git integration - Push modifications
      • Topology archive files upload/delete/edition (including topology yaml)
      • View modifications history on components and topologies
      • Save, Undo, Redo changes in the design view
      • Keyboard shortcuts
    • Workspace - Premium
      • Upload, modify components and topologies in a logical space
      • Share component from your workspace with users, groups and public
    • Security
      • Secure All http connections within Cloudify 3.4 – Premium

    Orchestrators certifications

    • Cloudify v3.4

    Infrastructure support

    • Azure (ARM) support (linux and windows compute, volumes, public IPs and network, security groups, H.A. policy) - Premium
    • Mesos / Marathon
    • Docker
      • Deploy container based topology relying on Mesos plugin or Cloudify Kubernetes plugin

    TOSCA

    • Support for alien 4 cloud 1.1 and 1.2 DSL
    • Support Tosca List and Map types at input level
    • Manage Artefact repositories - Premium

    Additional Premium features

    • v1.2 to v1.3 Migration tools
  • To be defined Future

    Core

    • Simple profile in YAML v1.0
    • On-demand hybrid deployment through Cloudify integration & user defined matching
    • Advanced post-deployment updates, upgrades & sync

    • Multi locations 24x7 app-centric placement & runtime policies
    • Workspaces & more Git integration

    More orchestrators, infrastructures & artefacts coverage

    Additional Premium features

    • Analytics and governance dashboards
    • Simplified Self Service
    • Social
    • Chargeback & billing platforms integration
    • Cloud brokering
